Secure Notes

A multi-user secure notes application built with Flask. Each user's notes are encrypted using keys derived from their own password.

Login Create Account

Features

  • Multi-user authentication system
  • Email verification before account activation
  • Secure password hashing
  • Envelope encryption for notes
  • User-specific encryption keys derived from password
  • Image cover support for notes
  • Clean responsive UI

Technologies Used

  • Flask
  • Jinja templating engine
  • Flask-Login for authentication
  • Flask-SQLAlchemy for database ORM
  • Flask-Mailman for email verification
  • Cryptography library for encryption
  • Bcrypt for password hashing

Security Design

This project implements envelope encryption where each user's notes are encrypted using a Data Encryption Key (DEK). The DEK itself is encrypted using a Key Encryption Key (KEK) derived from the user's password through a key derivation function.

Passwords are securely hashed and never stored in plaintext. User data is isolated and protected with multiple layers of security.
